# Conteúdo do Ecossistema

Este diretório recebe conteúdo Markdown versionado da Central de Conhecimento durante a implantação inicial.

O banco é responsável por publicação, permissões, vigência e auditoria. Arquivos neste diretório não devem ser lidos diretamente pelas interfaces de produção.

Use o modelo em [`_templates/conteudo.md`](_templates/conteudo.md) e siga [as regras editoriais](../docs/ecossistema/CENTRAL_DE_CONHECIMENTO.md).

## Organização

```text
knowledge/
  _templates/
  comercial/
  hoteis/
  produtos-experiencias/
  procedimentos/
  faq/
  marketing/
  ia/
  operacao/
  rh/
  financeiro/
  ti/
  academia/
```

Diretórios organizam autoria. Escopo e consumidor são definidos nos metadados, não pela posição física do arquivo.
